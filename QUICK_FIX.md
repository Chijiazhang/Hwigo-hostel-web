## 🚨 常见问题

### 1. 开发服务器未启动
**症状**: 访问任何页面都显示"无法访问此网站"
**解决**: 运行 `npm run dev`

### 2. 端口被占用
**症状**: 启动失败，显示端口3000被占用
**解决**: 使用 `npm run dev -- -p 3001`

### 3. 依赖缺失
**症状**: 启动时出现模块找不到的错误
**解决**: 运行 `npm install`

### 4. 浏览器缓存
**症状**: 修改代码后页面没有更新
**解决**: 硬刷新页面 (Ctrl+F5)
